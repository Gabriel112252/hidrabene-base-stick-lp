# Base Stick FPS 50 — Landing Page (rascunho)

Protótipo de landing page (mobile-first, responsivo) do lançamento da **Base Stick FPS 50** da Hidrabene Derma.

> ⚠️ **Conteúdo confidencial / pré-lançamento.** Página não indexada em buscadores (`noindex`). Compartilhe o link apenas internamente.

## Ver online
Hospedado via GitHub Pages — link na seção **Deployments / Pages** do repositório.

## Rodar local
É um arquivo estático. Basta abrir `index.html` no navegador (ou servir a pasta com qualquer servidor estático).

## Estrutura
- `index.html` — página completa (HTML + CSS + JS inline)
- `assets/` — imagens, vídeo (`video/`), logo e selos da marca (`brand/`)
